# Network-communication-in-aviation-in-Python

## Scenario
Effective communication in aviation ensures safe and efficient flight operations by facilitating real-time information exchange between pilots, air traffic controllers, and other personnel. This exchange includes sharing crucial details about weather, flight paths, clearances, and potential hazards, leading to reduced risks and improved operational efficiency. 

 # Question
Write a python program that checks the information of the in-Flight events and if the information are not in the specified safety margin, the server must send an email to alert the monitoring service that about the specs of the modem.
